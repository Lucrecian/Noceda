Proyecto: pagina de centro de estetica, en la que se pueden ver los diferentes tratamientos y servicios.
Estructura con Html, Css y Javascript
En cuanto a JavaScript se agrego en el la carpeta Scripts archivo Main.js un ciclo, en este caso el FOR para agendar turnos. También un condicional IF ELSE para ingresar el nombre de usuario del paciente. Y también funciones como para dar la bienvenida al ingrear al sitio y para ingresar clave.
arrays y objeto combinados 

Al cargar la página, se mostrará una sección que presenta los servicios disponibles del salón de belleza.

Los usuarios podrán revisar la lista de servicios y seleccionar los que deseen agregando un check en los servicios de su elección.

A medida que los usuarios seleccionan servicios, se irá actualizando el carrito de compras en la página, mostrando los servicios seleccionados y el total acumulado.

Los usuarios podrán vaciar el carrito de compras en cualquier momento utilizando el botón "Vaciar Carrito".

Cuando los usuarios estén listos para completar su compra, podrán hacer clic en el botón "Guardar" en el carrito de compras.

Se abrirá un formulario de datos del cliente, donde los usuarios deberán ingresar su nombre y correo electrónico.

Los usuarios deberán completar todos los campos requeridos en el formulario y luego hacer clic en el botón "Enviar".

Después de enviar el formulario, se mostrará un mensaje de confirmación indicando que la compra se realizó exitosamente.

El carrito de compras se reiniciará, listo para que los usuarios realicen nuevas compras si lo desean.
